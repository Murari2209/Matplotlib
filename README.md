pip install pandas matplotlib

## Task 1 – Line Plot (Sales Trend)

A line plot is created to visualize how total sales change over months.

Steps:
Convert Order Date to datetime
Extract Month
Calculate total sales per month
Plot a line chart using Matplotlib

## Task 2 – Scatter Plot

A scatter plot is used to show the relationship between numerical values.

Example used:
Order Index vs Sales


## Task 3 – Bar Plot

Two bar charts are created:

Vertical Bar Chart
Shows total sales by category.

Horizontal Bar Chart
Displays the same information but in horizontal form.

## Task 4 – Multiple Bar Plot

A multiple bar chart compares total sales across different years.

Steps:
Extract Year from Order Date
Group sales by year
Plot bars for each year

## Task 5 – Stacked Bar Chart

A stacked bar chart is used to display cumulative sales.

Example:
Sales stacked by year

## Task 6 – Histogram (Sales Distribution)

A histogram is used to visualize the distribution of sales values.

Steps:
Select numeric column Sales
Choose appropriate number of bins
Plot histogram

Task 7 – Pie Chart (Market Share)

A pie chart shows the proportion of sales by category.
